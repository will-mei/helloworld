汉字的Unicode编码

关于日常使用的汉字,一般使用2w基本汉字就够了

Unicode编码分布中,常用中文文字范围在 4E00-9FFF: 中日韩统一表意文字(其中尾部有部分空白码点,但是脚本中判断字符是否为中文的情况建议使用9FFF作为范围的结束点,因为该范围是相对完整的标准范围),如果是英文版查询结果则带有CJK(CJK = Chinese-Japanese-Korean)字样的说明信息,通常很多中文字符集和相关字体也都有cjk标识,下表是Unicode对常用中文的收录情况:


字符集字数Unicode编码
基本汉字20902字4E00-9FA5
基本汉字补充38字9FA6-9FCB
扩展A6582字3400-4DB5
扩展B42711字20000-2A6D6
扩展C4149字2A700-2B734
扩展D222字2B740-2B81D
康熙部首214字2F00-2FD5

兼容扩展542字2F800-2FA1D

汉字笔画36字31C0-31E3
