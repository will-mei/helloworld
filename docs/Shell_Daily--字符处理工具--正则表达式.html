正则表达式

命令行环境下正则表达式是字符处理的必备知识,可以快速匹配文本中的内容,简化命令的输入输出结果.

正则表达式(regular expression)描述了一种字符串匹配的模式（pattern），可以用来检查一个串是否含有某种子串、将匹配的子串替换或者从某个串中取出符合某个条件的子串等,包括普通字符（例如，a 到 z 之间的字母）和特殊字符（称为"元字符"）。


注意: 在不同的应用程序(grep,sed等)和语言环境(java,python,ruby等)中正则表达式的表示和匹配模式将稍有不同,即使同样的linux系统中,不同的应用程序对于正则表达式的支持也不同.

例如,在sed和awk中的正则表达式语法相对较为易于使用,也符合通用习惯,但是grep的表达式支持则比较简单,对于复杂表达式的匹配并不能直接打到想要的结果,有时需要反复转译才能匹配,可以使用grep -E参数或者-P参数使用Perl表达式代替,也可以使用增强版的egrep命令.

虽然正则表达式在不同环境的支持不同,但是相互之间大同小异,学习正则表达式关键在于理解常用的匹配模式,并不需要全部记忆,通常在需要时根据需求查询即可.
